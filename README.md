# Album Cover Grabber

A minimal tool to search and download high-res album covers.

I DJ, so I made this to quickly grab clean cover art for sets, captions, and visuals — no clutter, just search and download.

**Live:** https://player99labs.github.io/cover-grabber/

- Search by song + artist, pick the exact cover you want
- High-res downloads (up to 1000×1000)
- Batch mode: collect across searches, download all at once
- Single HTML file, no install, powered by the free Deezer API
